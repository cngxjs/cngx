# @cngx/common

Headless behavioral directives for Angular. Composition atoms and molecules built on Angular CDK.

## Entry Points

| Entry Point | Description |
|-|-|
| `@cngx/common/a11y` | ARIA and accessibility directives |
| `@cngx/common/interactive` | Click, gesture, navigation, and action directives |
| `@cngx/common/layout` | Layout observation, scroll, drawer, and skeleton directives |
| `@cngx/common/data` | Sort, filter, paginate, and data source utilities |

The primary `@cngx/common` re-exports all four entry points for backwards compatibility.

## Peer Dependencies

`@angular/core`, `@angular/common`, `@angular/cdk`
